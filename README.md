# cn-ihg-hub

## 项目简介

cn-ihg-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，而是作为一个通用存放空间，用于保存各类独立的 HTML 资源。

## 目录说明

```
cn-ihg-hub/
├── pages/          # 存放独立 HTML 页面
├── assets/         # 页面所需的资源文件（图片、样式、脚本等）
├── docs/           # 相关文档和说明
└── README.md       # 本文件
```

- `pages/`：核心目录，存放所有归档的独立 HTML 页面文件。
- `assets/`：若页面需要外部资源，可统一存放在此目录。
- `docs/`：存放与归档内容相关的文档、注释或参考材料。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，每个页面可单独部署或查看。归档内容主要涵盖各类静态页面，包括但不限于信息展示页、工具页面、演示页面等。

- 页面文件命名建议清晰、具备描述性。
- 每个页面应尽量自包含，减少对外部资源的依赖。
- 如需引用外部资源，建议将其存放于 `assets/` 目录中，并在页面内使用相对路径引用。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面时，请尽量保持目录结构整洁。
- 欢迎提交 Pull Request 贡献新的独立 HTML 页面，或对已有页面进行改进。
- 提交前请确保页面内容不包含恶意代码、侵犯他人权益或违反法律法规。
- 对于不再需要的页面，可在经过确认后移除。
